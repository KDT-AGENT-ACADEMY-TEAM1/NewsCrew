용어집 (Glossary)

작성 에이전트가 전문 용어를 정확히 설명하거나 각주를 달 때 참조한다.

A. 서비스 용어
- 키워드 구독: 구독자가 관심 분야를 키워드로 등록하면, 해당 주제 위주로 뉴스레터가 생성되는 기능.
- 검수 에이전트: 작성된 초안의 품질·규정 준수 여부를 자동 점검하는 에이전트. 기준 미달 시 작성 단계로 되돌린다.
- 휴먼 인 더 루프(Human-in-the-loop): 자동 생성 결과를 사람이 최종 확인·승인하는 절차. 승인 전까지 그래프 실행이 일시 중단된다.

B. 기술 용어
- LangGraph: 상태(State) 기반으로 여러 에이전트의 흐름과 분기를 그래프로 정의하는 프레임워크.
- Conditional Edge(조건부 엣지): 노드의 판정 결과에 따라 다음으로 이동할 노드를 동적으로 결정하는 연결.
- interrupt_before: 특정 노드 실행 직전에 그래프를 멈추는 기능. 사람의 승인 대기에 사용한다.
- State Checkpoint(상태 체크포인트): 그래프 실행 중간 상태를 저장해, 중단 후 같은 지점에서 재개할 수 있게 하는 장치.
- RAG(검색 증강 생성): 외부/사내 문서를 검색해 그 내용을 근거로 답변·문장을 생성하는 방식.
- ChromaDB: 문서 임베딩을 저장하고 의미 기반 유사도 검색을 지원하는 벡터 데이터베이스.
- 임베딩(Embedding): 텍스트를 의미를 담은 숫자 벡터로 변환한 표현. 유사도 검색의 기반이 된다.

C. 발송 관련 용어
- 옵트아웃(Opt-out): 구독자가 수신을 거부하는 것. 즉시 발송 목록에서 제외해야 한다.
- 광고성 정보: 영리 목적의 홍보·판촉 내용. 표시 의무와 발송 시간 제한이 적용된다.
